# CallVibe Docs

This documentation portal is built with Docusaurus.

## Installation

```bash
npm install
```

## Local Development

```bash
npm start
```

This starts a local server with hot reload.

## Build

```bash
npm run build
```

This generates static content in the build directory.

## Serve Build Locally

```bash
npm run serve
```

## Notes

- Content lives under docs/docs
- Sidebar is configured in docs/sidebars.ts
- Site config is in docs/docusaurus.config.ts
