# Categories

Categories are labels the AI assigns to every processed call. They are used to segment, filter, and report on calls by topic or intent.

## Category Types

- **System Default:** Created by CallVibe, apply across all tenants. Cannot be edited or deleted.
- **Custom:** Created by your team, tenant-specific. Can be edited and deleted.

## Fields

- **Name:** Short label shown on call cards and filters (e.g., "Hot Lead")
- **Description:** Tells the AI what calls belong to this category — write this clearly for better accuracy
- **Notification Criteria:** Optional plain-English rule that triggers an alert when matched (e.g., "Customer mentions cancellation or refund")

## Manage Categories

### Add a Category
1. Open Settings > Categories.
2. Click New Category.
3. Enter Name and Description, and optionally Notification Criteria.
4. Click Save.

### Edit a Category
Click the pencil icon next to any custom category, update the fields, and save.

### Delete a Category
Click the trash icon. Existing calls tagged with that category keep the label; new calls will no longer be assigned to it.
