# Notifications

## Slack

- Connect using OAuth from Settings > Notifications.
- Disconnect from the same screen when rotating workspaces.

## Email Digests

Configure recipients with:

- Email address
- Admin or agent scope
- Notification type (daily or weekly)
- Timezone
- Send time
- Enabled or disabled status

## Operational Tips

- Use admin digests for leadership summaries.
- Use agent-level digests for team coaching.
- Set timezone and send-time explicitly for predictable delivery.
