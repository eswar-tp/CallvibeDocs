# Team Management

## Invite a Team Member

1. Open Team.
2. Click Invite Team Member.
3. Enter email and choose role.
4. For agent role, map allowed agent names.
5. Send invitation.

## Roles

- Owner: Full tenant access
- Agent: Scoped visibility using mapped agent names

## Manage Existing Members

- Edit agent-name mappings
- Remove non-owner users
- Review pending invites

## Best Practices

- Keep a small owner group.
- Map agent names exactly as they appear in call data.
- Remove stale pending invites.
