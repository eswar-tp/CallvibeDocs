# 8x8 Integration

## Required Credentials

- API key
- Username
- Password
- API secret

## Setup

1. Open Settings > Integrations.
2. Select 8x8.
3. Enter all required credentials.
4. Click Connect.
5. Start worker.

![Settings page](/img/settings.JPG)

![8x8 connection](/img/8x8.JPG)

## Validation

- 8x8 status shows Connected.
- Worker status remains running.
- Imported calls are visible in analytics pages.
