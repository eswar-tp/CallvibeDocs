# Kaleyra Integration

## Required Credentials

- API key

## Setup

1. Open Settings > Integrations.
2. Select Kaleyra.
3. Enter API key.
4. Click Connect.
5. Start worker.

## Validation

- Card status shows Connected.
- Worker remains running.
- Call ingestion is visible in Calls.
