# SalesTrail Integration

## Required Credentials

- API key

## Setup

1. Open Settings > Integrations.
2. Select SalesTrail.
3. Enter API key.
4. Click Connect.
5. Start worker.

## Validation

- Integration status shows Connected.
- Worker runs without provider error.
- New calls appear in Calls view.
