# CallVibe Documentation

CallVibe helps your team capture calls and WhatsApp conversations, analyze sentiment and intent, and drive faster follow-ups.

Use this portal to:

- Onboard users and configure roles
- Connect provider integrations
- Understand platform navigation and workflows
- Configure notifications and customer rules
- Troubleshoot common operational issues

## Recommended Reading Path

1. Start with [Quickstart](./quickstart.md)
2. Continue with [Account and Access](./user-guide/account-and-access.md)
3. Set up providers from [Integrations Overview](./integrations/overview.md)
4. Review [Troubleshooting](./troubleshooting/common-issues.md)

## Scope

Current docs include:

- User onboarding and access
- Platform navigation
- Integration setup for each provider
- Admin operations
- API overview and webhook patterns
- Common troubleshooting playbooks
