# Account and Access

## Authentication Flows

CallVibe supports:

- Organization signup
- Email/password login
- Password reset and update
- Invitation-based team signup

## Roles

- Owner: Full tenant access, integrations, settings, team management
- Agent: Access limited to mapped agent names

## Invite and Join Flow

1. Owner sends invite from Team.
2. Invitee opens join link.
3. Invitee sets full name and password.
4. Account is activated and can sign in.

## Password Recovery

1. User requests reset from Reset Password.
2. User receives email link.
3. User sets new password on Update Password page.

## Access Tips

- Keep at least one owner account active.
- Use agent-name mapping to enforce data visibility.
- Review pending invites periodically.
