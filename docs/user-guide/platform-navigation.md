# Platform Navigation

The left sidebar is split into Main and Administration.

## Main

- Dashboard: KPI cards and top-level health
- Calls: Search, filter, and inspect call records
- WhatsApp: Chat threads and message analytics
- Customers: Customer-centric history and context
- Analytics: Trends, sentiment, categories, and patterns
- Action Items: Follow-up tasks from calls

## Administration

Visible to owner role.

- Agents: Agent-level summaries and details
- Team: User invites, role management, and mappings
- Settings: Integrations, Notifications, Customer Management, Categories

## Route Reference

- / for Dashboard
- /call-records for Calls
- /whatsapp for WhatsApp
- /customers for Customers
- /analytics for Analytics
- /action-items for Action Items
- /agents for Agents
- /users for Team
- /settings and /settings/* for configuration sections
